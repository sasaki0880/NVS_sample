Overview of Speech

This page contains three main sections:

  • Original Speech: contains the raw speech samples.
  • Noise Vocoded Speech(NVS) (without MFB, 0-64 Hz): contains the noise-vocoded speech of original speech, synthesized without using Modulation Filterbank.
  • NVS (with MFB, 0-64 Hz): contains the noise-vocoded speech of original speech, synthesized using Modulation Filterbank.
  • NVS (with MFB, 4-16 Hz): contains the noise-vocoded speech of original speech, with the modulation frequency components in the 4-16 Hz band.
  • NVS (with MFB, 4-16 Hz, Time-reversed): contains the noise-vocoded speech of original speech, with time-reversed temporal amplitude envelope.
  • NVS (with MFB, 4-16 Hz, Time-stretched): contains the noise-vocoded speech of original speech, with time-stretched temporal amplitude envelope.
  • NVS (with MFB, 4-16 Hz, Time-compressed): contains the noise-vocoded speech of original speech, with time-compressed temporal amplitude envelope.
  • NVS (with MFB, 4-16 Hz, Time-stretched and compressed): contains the noise-vocoded speech of original speech, with time-stretched and compressed temporal amplitude envelope.
